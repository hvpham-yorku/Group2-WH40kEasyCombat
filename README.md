This is a Project from the course EECS 2311 in York University. (Winter 2026)

ProjectDescription:
A software designed to assist in Warhammer 40,000 battles, offering various scoring and
support functions to make combat easier.

Eclipse running configuration
1. Going to src\main\java\eecs2311\group2\wh40k_easycombat
2. Right click Main.Java
3. Run As --> Run configurations --> Maven Build
4. Enter  ···clean javafx:run···  to Goals
5. Apply and Run
